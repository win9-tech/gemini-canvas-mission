# 웹 백엔드 레벨1 - 미션: 사용자가 있는 Gemini 웹앱 출시하기

Gemini Canvas를 활용해 제작한 4개의 웹앱입니다.

---

## 🍽️ 찔까 말까 (유틸리티 앱)

우테코 식단표를 보고 매번 뭘 먹을지 고민하는 시간을 줄이기 위해 만든 앱입니다.  
식단표 사진을 올리면 AI가 메뉴를 자동 추출하고, 다이어트/증량 목표에 맞는 최적 식단을 추천해줍니다.

**핵심 AI 기능:** 이미지 기반 메뉴 자동 추출 / 목표 맞춤 식단 분석 및 추천

[배포 링크](https://gemini.google.com/share/866c324d1381)

---

## 👑 가위바위보의 신 (게임)

쉬는 시간에 아무 생각 없이 짧게 즐길 수 있는 코인 베팅형 가위바위보 게임입니다.  
연승할수록 신의 분노 확률이 올라가고, 장갑을 구매해 승률과 분노 저항을 높일 수 있습니다.

**핵심 메커니즘:** 코인 베팅 / 연승 리스크 시스템 / 장갑 상점

[배포 링크](https://gemini.google.com/share/aa11ac0c2d9e)

---

## 💻 백엔설 (학습 앱)

백엔드 개념을 공부한 뒤 퀴즈로 점검하는 앱입니다.  
학습 주제를 입력하면 AI가 주제 적합성을 검증하고, 난이도별 퀴즈를 자동 생성해줍니다.

**핵심 AI 기능:** 주제 유효성 검증 / 난이도 맞춤 퀴즈 자동 생성

[배포 링크](https://gemini.google.com/share/2457aca00892)

## 🤝 네트워코 (페어 프롬프트 릴레이)

페어 @johnprk, @win9-tech 와 함께 만든 크루 네트워킹 매칭 앱입니다.  
MBTI와 성향 데이터를 AI가 분석해 단순 랜덤이 아닌 데이터 기반으로 크루를 매칭하고, 맞춤 대화 주제까지 제안합니다.

**핵심 AI 기능:** 성격 기반 매칭 / 매칭 사유 생성 / 맞춤형 대화 주제 제안
